# JFrog Configuration

This directory will contain JFrog-specific configuration files for:

- JFrog CLI configuration
- Xray policy definitions
- Artifactory repository configurations
- Build info templates

## Future Integration Points

1. **JFrog CLI**: Commands for build-collect-env, build-publish, build-scan
2. **Xray**: Security and license scanning policies
3. **Artifactory**: Repository configurations and permissions
4. **Curation**: Package curation policies

To be implemented after MVP.
